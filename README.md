**PROJECT TITLE:** 

BIKE SHARING PREDICTOR APP

**DESCRIPTION:**

This is an interactive app that visualizes weather forecast data and predicts hourly bike-sharing demand based on the weather for these cities: New York, USA, Paris, France, Suzhou, China, London. UK and Seoul, South Korea

This app also contains a Leaflet based interactive map that shows the max predicted bike-sharing demand in the next 5 days. 

This app relied heavily on information from a linear prediction model file (model_prediction.r) file to parse the information reflected therein 

**Features include:** 

•	a select input (dropdown) to select a specific city

•	a line chart showing the temperature trend in the next 5-days.

•	A line chart showing the bike-sharing demand prediction trend

•	A scatter plot showing the correlation between humidity and bike demand

**Technologies used:**
•	RStudio: this is a well known powerful tool for the use of R programming language

•	Leaflet: the leaflet package is also well know for being a powerful tool for creating interactive maps which was useful in this situation.

**Credits:**

Weather information was gotten from the website, **“OpenWeather.com”** which provides global weather data APIs for current conditions, forecasts, historical archives, air quality, maps, and industry-specific use cases.
